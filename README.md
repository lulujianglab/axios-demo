# axios-demo

www 文件存放了多个静态文件来编写 `axios` 封装方法的 demo

静态文件直接引用了 `vuejs` 进行开发

且通过 `express` 编写简易服务端代码

## 项目编译和运行

``` bash
1) 安装依赖

> npm install

2) 运行脚本

> node server.js

3) 浏览器端访问页面

> http://localhost:8888/index.html 等
```

更多代码介绍可以查看 [axios 部分封装方法的解析与实践](https://github.com/lulujianglab/blog/issues/44)
